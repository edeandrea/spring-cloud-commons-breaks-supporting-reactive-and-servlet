# spring-cloud-commons-breaks-supporting-reactive-and-servlet
Sample project for https://github.com/spring-cloud/spring-cloud-commons/issues/357

If you run the tests you'll notice that [ReactiveTests](src/test/java/com/example/reactive/ReactiveTests.java) fails with the following exception. If you remove the [`compile 'org.springframework.cloud:spring-cloud-context:2.0.0.M9'` dependency from build.gradle](build.gradle#L32) then the tests run fine.
